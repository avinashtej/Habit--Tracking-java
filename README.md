 # Habit Tracking Java Application

A menu-driven Habit Tracker application developed using Java and Object-Oriented Programming principles.

## Features
- Add, edit, delete habits
- Daily and weekly habit tracking
- Streak management
- Reminder scheduling using multithreading
- Weekly and monthly progress reports
- Persistent storage using serialization

## Concepts Used
- Abstraction
- Inheritance
- Interfaces
- Exception handling
- Multithreading
- File handling
- Serialization

## How to Run
javac HabitTrackerApp.java  
java HabitTrackerApp
